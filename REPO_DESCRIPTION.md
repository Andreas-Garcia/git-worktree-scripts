# Repository Description

## Short Description (for GitHub/GitLab)

Use this as the repository description field (one-liner):

```
Bash scripts for managing git worktrees: create, open, and remove worktrees with automatic editor integration and repository-specific setup hooks.
```

## Longer Description (for README or About section)

```
A collection of bash scripts that automate git worktree management across multiple repositories. Create worktrees from main, open existing ones interactively, and safely remove them with merge detection. Supports Cursor and VS Code, with hooks for repository-specific setup (Python venv, npm install, etc.). Install once, use everywhere.
```

## Key Points

- **What it is**: Bash scripts for git worktree management
- **What it does**: Automates creating, opening, and removing git worktrees
- **Key features**: Editor integration, merge detection, repository-specific setup hooks
- **Use case**: Developers working on multiple features simultaneously
- **Installation**: One-time setup, reusable across all repositories
